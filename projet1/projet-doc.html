Documentation Technique

Objectif du projet

Installer et configurer un système Debian 11 sur une machine virtuelle, installer les logiciels de base nécessaires et mettre en place un environnement de travail fonctionnel.

Étapes réalisées

  • Téléchargement de l'ISO officiel Debian 11
  • Création de la machine virtuelle
  • Installation pas à pas de Debian 11
  • Configuration des utilisateurs et de l'accès root
  • Installation de Firefox et des outils supplémentaires
  • Vérification de la bonne configuration réseau

Commandes importantes

sudo apt update
sudo apt upgrade
sudo apt install firefox-esr

Compétences acquises

  • Installation système complète
  • Configuration réseau et utilisateurs
  • Utilisation des gestionnaires de paquets apt
  • Travail en autonomie
