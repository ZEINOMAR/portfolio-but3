Documentation Technique

Objectif du projet

Créer une base de données relationnelle autour du naufrage du Titanic, incluant la conception du modèle entité-association, la création des tables SQL, l’insertion des données et la production d’un compte-rendu technique en anglais.

Étapes réalisées

  • Recherche d'informations pertinentes sur le naufrage
  • Spécification des exigences fonctionnelles
  • Modélisation entité-association (MERISE)
  • Implémentation des tables SQL
  • Insertion et structuration des données
  • Rédaction d'un compte-rendu technique (EN)

Exemple de commandes SQL utilisées

CREATE TABLE Passagers (
  id INT PRIMARY KEY,
  nom VARCHAR(100),
  age INT,
  classe VARCHAR(10),
  survivant BOOLEAN
);

INSERT INTO Passagers (id, nom, age, classe, survivant)
VALUES (1, 'John Smith', 32, '1ère', TRUE);

Compétences acquises

  • Modélisation de base de données (EA/SEA)
  • SQL : création de tables, insertion, requêtes simples
  • Structuration et normalisation des données
  • Communication technique en anglais
  • Travail en équipe et rigueur dans la documentation
