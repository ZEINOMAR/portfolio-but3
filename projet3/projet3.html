Projet 3 : Création base de données Titanic

Conception et implémentation d'une base de données complète autour du naufrage du Titanic en SQL et modélisation SEA.

SQL Base de données Modélisation Titanic

À propos du projet

Durant ce projet, nous avons été chargés de développer une base de données sur le Titanic. Après une phase de recherche d'informations sur le naufrage, nous avons élaboré les spécifications et conçu le modèle entité-association pour structurer la base.

Nous avons ensuite créé les différentes tables SQL et inséré les données en veillant à assurer la cohérence et l'intégrité des informations.

Enfin, j'ai rédigé un compte rendu en anglais résumant le projet, ses objectifs et les résultats obtenus.

Ce projet m'a permis de maîtriser la gestion de données, la modélisation et l'implémentation d'une base SQL complète et structurée.

Compétences développées :

  • Modélisation entité-association (MERISE, SEA)
  • Rédaction de cahier des charges
  • Création de tables SQL et insertion de données
  • Optimisation de la structure relationnelle
  • Production de documentation technique en anglais
