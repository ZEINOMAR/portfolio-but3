Documentation Technique

Objectif du projet

Développer une application complète de gestion de tournois LAN (esports), depuis la modélisation fonctionnelle jusqu’au développement et à l’implémentation de l’interface utilisateur.

Étapes réalisées

  • Analyse des besoins fonctionnels et contraintes
  • Création des diagrammes UML (cas d’utilisation, classes, séquences)
  • Conception de la maquette non-fonctionnelle
  • Développement de l’application en Java et JavaFX
  • Implémentation des contraintes algorithmiques pour la gestion des tournois
  • Tests fonctionnels et validation finale

Technologies utilisées

  • Java
  • JavaFX (UI)
  • UML (modélisation)
  • Git / GitHub (gestion de version collaborative)

Compétences acquises

  • Gestion complète de projet logiciel
  • Conception UML détaillée
  • Programmation orientée objet avancée
  • Développement d’interface utilisateur en JavaFX
  • Travail collaboratif et gestion d’équipe
