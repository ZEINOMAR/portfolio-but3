Documentation Technique

Objectif du projet

Réaliser un site web à destination des élèves de 3ème recherchant un stage en entreprise. Le projet comprend un recueil de besoin, la conception d’un wireframe et la réalisation partielle des pages du site.

Étapes réalisées

  • Réalisation d’une carte mentale pour organiser les besoins utilisateurs
  • Spécification du cahier des charges fonctionnel
  • Conception collaborative du wireframe
  • Répartition des tâches entre les membres de l'équipe
  • Développement HTML et mise en forme CSS

Compétences acquises

  • Recueil et analyse des besoins utilisateurs
  • Création de wireframes fonctionnels
  • Travail collaboratif sur un projet web
  • Développement HTML & CSS structuré
  • Gestion de projet et respect des délais
