Projet 11 : Stage chez POMA

Amélioration d’outils internes chez POMA : optimisation d’API, refonte d’interfaces et développement d’automatisations de processus métier.

Python API Web JavaScript UX/UI

À propos du projet

Lors de mon stage chez POMA, j’ai participé à la modernisation de plusieurs outils internes. J’ai optimisé des scripts Python générant des tableaux de données, amélioré l’API et automatisé plusieurs tâches métiers. J’ai également repensé des interfaces web existantes en HTML, CSS et JavaScript, en y ajoutant un loader, des champs de saisie automatisés ainsi qu’une barre de recherche dynamique pour faciliter la navigation.

Ce stage m’a permis d’élargir mes compétences techniques tout en découvrant les différentes activités industrielles de l’entreprise grâce à des formations internes et des visites de sites.

Compétences développées :

  • Optimisation de scripts Python et gestion d’API
  • Refonte d’interfaces utilisateur avec HTML, CSS, JavaScript
  • Intégration de fonctionnalités de recherche et d’automatisation
  • Travail en environnement industriel collaboratif
  • Organisation, gestion de projet et communication avec les équipes métiers
