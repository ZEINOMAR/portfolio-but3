Documentation Technique

Objectif du projet

Installer et configurer un serveur Debian 11 complet avec Apache, PostgreSQL et PHP. Documenter l'intégralité du processus pour pouvoir reproduire facilement l'installation.

Ressources officielles

  • Documentation officielle Debian 11 (Bullseye)
  • Télécharger l’image ISO Debian 11 (DVD amd64)

Étapes réalisées

  • Téléchargement et vérification de l'image ISO Debian 11
  • Création de la machine virtuelle
  • Installation du système Debian
  • Installation et configuration de Apache, PHP et PostgreSQL
  • Création de la documentation avec captures d'écran

Commandes principales utilisées

# Mise à jour du système
sudo apt update
sudo apt upgrade

# Installation d'Apache
sudo apt install apache2

# Installation de PHP
sudo apt install php libapache2-mod-php

# Installation de PostgreSQL
sudo apt install postgresql postgresql-contrib

# Vérification des services
systemctl status apache2
systemctl status postgresql

Compétences acquises

  • Administration Linux Debian
  • Installation de serveur web Apache et PHP
  • Gestion de base de données PostgreSQL
  • Documentation technique en anglais
  • Automatisation et configuration système
