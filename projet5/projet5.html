Projet 5 : Guide d'installation serveur Debian 11

Installation et configuration d’un serveur Debian 11 complet avec Apache, PostgreSQL, PHP et documentation technique en anglais.

Debian 11 Apache PostgreSQL PHP Linux

À propos du projet

Dans ce projet, j'ai effectué une installation complète et documentée d'un serveur Debian 11 sur une machine virtuelle :

  • Téléchargement et vérification de l'image ISO Debian 11
  • Installation de Debian sur la VM
  • Configuration complète du serveur : Apache, PostgreSQL et PHP
  • Rédaction d’un guide technique illustré en anglais

Ce projet m’a permis de renforcer mes compétences d’administration système et de rédaction technique.

Compétences développées :

  • Administration Linux Debian
  • Installation serveur web Apache et PHP
  • Gestion de bases de données PostgreSQL
  • Documentation technique en anglais
  • Automatisation et configuration système
